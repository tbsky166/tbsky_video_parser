# 视频解析器

一个简单易用的视频解析工具，支持多种解析接口，帮助用户解析和播放各大视频平台的视频内容。

## 功能特点

- 🎯 **简洁界面**：采用坐标布局，界面清晰美观
- 🔧 **多接口支持**：集成多个视频解析接口
- ⚡ **快速解析**：输入网址即可快速解析播放
- 🎬 **内置播放**：支持内嵌网页播放，无需跳转外部浏览器
- ⌨️ **快捷操作**：支持回车键快速解析
- 🎨 **美观样式**：采用现代化UI设计，支持主题样式

## 支持的解析接口

- **夜幕解析**
- **虾米解析**

## 安装要求

### 系统要求
- Windows 操作系统
- Python 3.6 或更高版本

### 依赖库安装
```bash
pip install -r requirements.txt
```

依赖库包括：
- `pywebview` - 内嵌网页浏览器
- `Pillow` - 图像处理支持

## 使用方法

1. **运行程序**
   ```bash
   python main.py
   ```
   ```cmd
   python main.py
   ```

2. **选择解析接口**
   - 从下拉菜单中选择合适的解析接口
   - 默认选择"夜幕解析"

3. **输入视频网址**
   - 在输入框中粘贴要解析的视频网址
   - 支持各大主流视频平台的视频链接

4. **开始解析**
   - 点击"🎬 解析视频"按钮
   - 或直接按回车键开始解析

5. **观看视频**
   - 程序会自动打开内嵌播放器
   - 播放窗口支持调整大小和全屏

## 界面布局

程序采用坐标布局，各组件位置如下：
- **标题**：顶部居中显示
- **选择解析接口**：左对齐，包含下拉选择框
- **网址输入框**：在下拉菜单下方，按钮上方
- **解析按钮**：在输入框下方
- **官网链接**：右下角位置

## 快捷键

- `Enter` / `回车键`：快速解析当前输入的网址
- `Esc`：退出播放窗口（在播放界面）

## 注意事项

- 请确保输入完整的视频网址（包含http://或https://）
- 某些视频平台可能需要特定的解析接口才能正常工作
- 解析速度和效果取决于所选接口的服务状态
- 建议在稳定的网络环境下使用以获得最佳体验

## 故障排除

### 无法解析视频
- 检查输入的网址是否正确
- 尝试切换不同的解析接口
- 确认网络连接正常

### 播放窗口无法打开
- 检查是否已安装所有依赖库
- 尝试重新运行程序
- 查看控制台输出获取详细错误信息

### 程序崩溃或无响应
- 确保系统满足运行要求
- 尝试以管理员身份运行
- 检查防火墙设置是否阻止程序访问网络
